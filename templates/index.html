Roteamento Dinâmico no Flask

Aprenda como o Flask captura parâmetros diretamente da URL usando o decorador @app.route

🔍 Como funciona?

No Flask, o decorador @app.route('/saudacao/<nome>') define uma rota dinâmica. O trecho entre < > é um parâmetro variável que o Flask captura automaticamente o valor presente na URL e o passa como argumento para a função Python correspondente.

👩
/saudacao/Ana
nome = "Ana"
👨
/saudacao/Johnny
nome = "Johnny"
🤖
/saudacao/Python
nome = "Python"

📂 Arquivos do Projeto

Código Python usado no projeto:

main.py 
import os
from flask import Flask, render_template

# Criamos a instância da aplicação Flask.
# __name__ informa ao Flask onde procurar templates.
app = Flask(__name__)

# O decorador @app.route define qual URL aciona esta função.
@app.route("/")
def index():
    """Rota raiz: exibe instruções de como usar a demo."""
    return render_template("index.html")

# O decorador @app.route define a rota dinâmica.
# O trecho <nome> entre < > é um PARÂMETRO VARIÁVEL.
@app.route("/saudacao/<nome>")
def saudacao(nome):
    """ Rota dinâmica: recebe o parâmetro 'nome' da URL. """
    return render_template("saudacao.html", nome=nome)

if __name__ == "__main__":
    port = int(os.environ.get("PORT", 5000))
    app.run(host="0.0.0.0", port=port, debug=True)

💡 Conceitos-chave

  • @app.route(): decorador que registra uma URL como rota no Flask
  • <nome>: parâmetro variável capturado da URL em tempo de execução
  • def saudacao(nome): o argumento recebe automaticamente o valor da URL
  • Tipos suportados: <int:id>, <float:valor>, <path:caminho>
